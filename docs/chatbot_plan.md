# 네이버 클로바 챗봇 & 톡톡 연동 계획서

스튜디오생일 고객 상담을 자동화하기 위해 **CLOVA Chatbot**을 구축하고 **네이버 톡톡**에 연동합니다. 반복되는 예약 문의, 위치 안내, 가격 정보 등을 챗봇이 먼저 응대하도록 설정합니다.

## 📋 사전 준비 사항
- [ ] 네이버 클라우드 플랫폼(NCP) 계정 및 결제 수단 등록
- [ ] 네이버 톡톡 파트너 센터 계정 생성 (검수 완료 필요)

## 🛠️ 구축 단계

### 1. CLOVA Chatbot 도메인 생성 (NCP 콘솔)
- **도메인 생성**: '스튜디오생일_상담봇' 도메인 생성
- **대화 모델 설계**:
  - **인사말**: 고객 인입 시 첫 메시지 설정
  - **핵심 대화**: 예약 방법, 오시는 길(지도), 가격표, 원본/보정본 수령 안내 등
  - **폴백(Fallback)**: 이해하지 못한 질문 시 상담원 연결 안내

### 2. 네이버 톡톡 연동 설정
- **API Gateway 연동**: 챗봇 호출을 위한 Invoke URL 생성
- **톡톡 파트너 센터 설정**:
  - `개발자도구 > 챗봇 API 설정`에서 Invoke URL 등록
  - Channel Access Token 발급 및 클로바 빌더에 등록
- **메신저 연동**: 클로바 빌더 내 [메신저 연동] 메뉴에서 네이버 톡톡 활성화

### 3. 지식 베이스(FAQ) 학습
- 기존 상담 내역을 기반으로 FAQ 리스트 업로드
- 유사 질문(Utterance) 학습을 통해 인식률 향상

## 🧪 검증 계획
- [ ] **TalkTalk 테스트 전송**: 톡톡 메시지 창에서 질문 시 챗봇이 정상 응답하는지 확인
- [ ] **지도/링크 버튼 작동**: 위치 안내 시 네이버 지도 링크가 정상적으로 노출되는지 확인
- [ ] **상담원 전환**: 챗봇이 답변하기 어려운 경우 상담원 전환 안내가 나가는지 확인

## ⚠️ 주의 사항
- 톡톡 계정 검수는 보통 영업일 기준 1일 정도 소요됩니다.
- 챗봇 호출 횟수에 따라 NCP 과금이 발생할 수 있습니다. (무료 한도 확인 필요)
