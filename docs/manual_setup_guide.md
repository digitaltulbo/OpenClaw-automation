# 클로바 챗봇 직접 구축 & 연동 가이드

사용자께서 직접 정확한 사실에 기반한 대화 모델을 구축하실 수 있도록 단계별 액션 플랜을 정리해 드립니다. 

## 1단계: 대화 모델(FAQ) 직접 구축
현재 열려 있는 **챗봇 빌더(대화 목록)** 메뉴에서 아래 항목들을 하나씩 완성해 주세요.

- [ ] **인사말 설정**: 고객 인입 시 첫 응대 메시지 (예: "안녕하세요! 스튜디오생일입니다...")
- [ ] **핵심 FAQ 등록**:
  - **위치**: 야탑역 1번 출구, 성남대로916번길 7 (네이버 지도 링크 포함)
  - **가격**: 상품별 정확한 단가 (홈페이지 `bdaystudio.store/price` 참조)
  - **예약**: 네이버 예약 링크 안내
- [ ] **테스트**: 우측 하단의 [테스트] 버튼을 눌러 답변이 의도대로 나오는지 확인

## 2단계: 네이버 톡톡 연동 (기술 설정)
직접 입력하신 대화 모델이 준비되면, 이제 실제 톡톡과 연결해야 합니다.

1. **Invoke URL 복사 (클로바 빌더)**:
   - `설정 > 메신저 연동 > 네이버 톡톡 > 연동` 버튼 클릭
   - `APIGW 수동 연동` 항목을 열어 **Invoke URL**을 복사합니다.
2. **톡톡 파트너 센터 설정**:
   - [네이버 톡톡 파트너 센터](https://partner.talk.naver.com/) 접속
   - `개발자 도구 > 챗봇 API 설정` 이동
   - 복사한 **Invoke URL**을 '이벤트 받을 URL'에 붙여넣기
   - **Service 정보**에서 발급된 토큰(Authorization 값)을 복사합니다.
3. **토큰 등록 (클로바 빌더)**:
   - 클로바 빌더의 톡톡 연동 화면 아래쪽에 복사한 토큰을 붙여넣고 **[연동]** 클릭

## 3단계: 서비스 배포
- [ ] 빌더 상단의 **[학습 수행]** 버튼 클릭 (모델 학습)
- [ ] 학습 완료 후 **[서비스 배포]** 버튼 클릭 (실제 적용)

---
> [!IMPORTANT]
> **직접 작업하시면서 막히는 부분이 있다면 알려주세요.** 특히 2단계의 URL 복사나 토큰 등록 과정에서 기술적인 도움이 필요하시면 제가 즉시 가이드해 드리겠습니다.
